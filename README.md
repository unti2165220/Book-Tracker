# Book-Tracker 

Краткое описание:
Автор Юнкус Макар
Описание: test.py111 - это GUI-приложение для учета прочитанных книг. Позволяет сохранять данные в JSON, фильтровать список по жанру и объему 
Инструкция к запуску программы
1. Установите Python 3.x.
2. Скачайте проект: `git clone https://github.com`
3. Запустите программу: `python main.py`
    Тестирование
 Попробуйте оставить поле пустым — программа выдаст предупреждение.
 Введите текст в поле "Страницы" — программа потребует число.
 Добавьте книгу где-то 300 страниц и используйте фильтр.
